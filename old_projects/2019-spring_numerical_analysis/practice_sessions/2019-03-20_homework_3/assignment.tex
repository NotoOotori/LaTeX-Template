    \subsection{比较}

    三种方法求出的结果很接近, 误差在\matlabinline{1e-13}的水平.
    因为追赶法针对系数矩阵为三对角矩阵的情况经过了优化,
    所以明显追赶法要比其它两种方法来得快.

    \section{Vandermonde线性方程组的求解}
    
    代码如下:

    \lstinputlisting[
        caption=na3\_2.m,
        style=MATLAB-editor,
        basicstyle=\mlttfamily\scriptsize
    ]{na3_2.m}

    解得条件数分别为
    \begin{lstlisting}[
        caption=answer,
        style=MATLAB-editor,
        basicstyle=\mlttfamily\scriptsize,
        numberstyle={\color{black!33}\scriptsize\sffamily}
    ]
        1.104400098300552e+03
        5.594498784456730e+06
        2.273909810062094e+10
    \end{lstlisting}
    故随$n$增大, 条件数快速发散到无穷.

    $n=5$时, 方程组的解为
    \begin{lstlisting}[
        caption=answer,
        style=MATLAB-editor,
        basicstyle=\mlttfamily\scriptsize,
        numberstyle={\color{black!33}\scriptsize\sffamily}
    ]
        0.999999999998386
        1.000000000009673
        0.999999999976879
        1.000000000027570
        0.999999999983586
        1.000000000003906
    \end{lstlisting}

    加\matlabinline{1e-4}的小扰动后, 得到解为

    \begin{lstlisting}[
        caption=answer,
        style=MATLAB-editor,
        basicstyle=\mlttfamily\scriptsize,
        numberstyle={\color{black!33}\scriptsize\sffamily}
    ]
        1.083333333316156
        0.500000000104855
        2.195833333078592
        -0.424999999692060
        1.846033333148086
        0.799800000044371
    \end{lstlisting}

    可见条件数大的情况下, 即使是小扰动也会对方程组的解造成较大的影响.
    因为插值函数存在定理中的方法系数矩阵为Hilbert矩阵, 条件数很大,
    因此不用该方法求解插值函数.
